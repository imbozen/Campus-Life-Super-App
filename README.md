# Campus Life Super App

## Description
This project is a multi-page web application designed to improve campus life by providing students with dining options and campus events in one place.

## Features
- Dining page with:
  - Breakfast, lunch, dinner categories
  - Vegan and gluten-free filters
  - Voting system for meals
  - Ability to suggest new meals
- Events page with campus sporting events
- API integration using TheMealDB
- Local storage to save meals and votes

## Technologies Used
- HTML
- CSS
- Bootstrap
- JavaScript
- LocalStorage
- External API (TheMealDB)

## How to Run
1. Clone the repository
2. Open index.html in a browser
